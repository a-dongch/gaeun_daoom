# 가은다움 뷰티 - 프리미엄 반영구 화장 전문샵

## 프로젝트 소개

가은다움 뷰티는 8년 경력의 PMU Master Gaeun이 운영하는 프리미엄 반영구 화장 전문샵의 공식 소개 웹사이트입니다. 
헤어스트록 전문 기법으로 자연스럽고 개인 맞춤형 디자인을 제공하며, 4개 국어(한국어, 영어, 일본어, 중국어)로 서비스를 제공합니다.

### 주요 특징
- ✨ **4개 국어 지원**: 한국어, 영어, 일본어, 중국어 실시간 번역
- 🎨 **반응형 디자인**: 모바일, 태블릿, 데스크톱 완벽 대응
- 🏆 **8년 경력**: 4000+ 시술, 만족도 99%
- 💎 **프리미엄 UI/UX**: 고급스럽고 우아한 디자인
- 🌟 **부드러운 애니메이션**: 스크롤 및 전환 효과

## 현재 완성된 기능

### ✅ 구현 완료
1. **다국어 시스템**
   - 한국어, 영어, 일본어, 중국어 완전 번역
   - 언어별 폰트 자동 적용 (Noto Sans 시리즈)
   - localStorage를 통한 언어 설정 저장
   - 실시간 언어 전환 (페이지 새로고침 불필요)

2. **주요 섹션**
   - Hero 섹션: 브랜드 소개 및 핵심 통계
   - About Master: 대표원장 프로필 및 자격증
   - Philosophy: 5가지 반영구 철학
   - Mission: 브랜드 미션 및 비전
   - Premium Care: 4단계 케어 시스템
   - Specialization: 전문 서비스 항목
   - Footer: 브랜드 정보

3. **인터랙션**
   - 스크롤 애니메이션 (Intersection Observer)
   - 통계 숫자 카운트 업 애니메이션
   - 부드러운 스크롤 전환
   - 호버 효과 및 마이크로 인터랙션

4. **디자인 시스템**
   - 브랜드 컬러: #8B7355 (Primary), #C4A07A (Secondary)
   - 일관된 타이포그래피
   - 카드 기반 레이아웃
   - 반응형 그리드 시스템

- **Font Awesome 6.4.0**: 아이콘

## 핵심 컨텐츠

### 가은다움의 5가지 철학
1. **시간이 지나도 자연스럽게** - 시간이 지나도 자연스럽게 어울리는 눈썹
2. **편안하게 스며드는** - 과하게 진하지 않은, 편안한 눈썹
3. **조화되는 자연 디자인** - 얼굴에 부담 없는 디자인
4. **시간이 흐를수록 편안해지는** - 지속적인 만족감
5. **본래 내 눈썹 같은** - 자연스러운 결과

### 1:1 프리미엄 케어 4단계
1. **STEP 1**: 상담 및 예약 - 피부 상태와 원하는 스타일 파악
2. **STEP 2**: 맞춤 디자인 - 얼굴형, 직업, 라이프스타일 고려
3. **STEP 3**: 섬세한 시술 - 피부 손상 최소화 헤어스트록 기법
4. **STEP 4**: 사후 관리 - 지속적인 관리와 체크

### 자격 및 수상 경력
- 미용사 면허증 보유
- 피부관리/메이크업/네일아트 국가 자격증
- 컬러리스트 산업기사 자격증
- 서울 인터내셔널 뷰티 콘테스트 메이크업 부문 2위
- 국제뷰티엑스포 기능대회 반영구 부문 최우수상
- K뷰티전문가연합회 수석 심사위원
- 국제바디아트콘테스트 수석 심사위원

### 커스터마이제이션 가이드

#### 색상 변경
`css/style.css` 파일의 `:root` 섹션에서 CSS 변수를 수정:
```css
:root {
    --primary-color: #8B7355;    /* 주 색상 */
    --secondary-color: #C4A07A;  /* 보조 색상 */
    --accent-color: #D4AF75;     /* 강조 색상 */
}
```

#### 번역 추가/수정
`js/translations.js` 파일에서 각 언어별 객체를 수정:
```javascript
const translations = {
    ko: { /* 한국어 번역 */ },
    en: { /* 영어 번역 */ },
    ja: { /* 일본어 번역 */ },
    zh: { /* 중국어 번역 */ }
};
```

#### 새 언어 추가
1. `translations.js`에 새 언어 객체 추가
2. `index.html`의 언어 선택기에 버튼 추가
3. `style.css`에 폰트 패밀리 추가
